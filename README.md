## What this does

CS2 Profile Injector is built for Windows users who want a practical way to manage Counter-Strike 2 profiles, create configs, and keep gameplay presets organized through a local workflow. It is intended for editing profile data, moving settings in and out of setups, and consolidating common game options in a single place.

Rather than focusing on broad automation, the project is centered on profile customization and script-supported configuration. It includes skin-focused presets, crosshair preset management, and movement helpers such as bunnyhop-style scripts, with ongoing work aimed at config handling and API-assisted content generation.
